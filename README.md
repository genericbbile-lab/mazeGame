#Maze Game

#This is a game where you navigate a randomized maze and collect coins. There are two hidden monsters that will try to get you.

#Radar-8 coins: allows you to see where the monsters are for a turn(Button: r)
#Trap-3 coins: place down to stun a monster for 3 turns(Button: t)
#Teleport-6 coins: Teleport back 5 turns(Button: g)
